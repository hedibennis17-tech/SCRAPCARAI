# **App Name**: CarWizardAI

## Core Features:

- Voice Assistant: Interact with users through voice, understand their requests, and guide them through the car assessment process. Use a tool to intelligently respond or route.
- Client Information Collection: Collect and validate user information such as name, phone number and email.
- Vehicle Information Gathering: Collect data about the vehicle, including make, model, year, and any existing issues.
- Vehicle Condition Diagnostics: Evaluate the condition of the vehicle based on user inputs and technical questions; summarize the vehicle condition and allow for confirmation or corrections.
- Automated Valuation: Estimate the value of the vehicle based on its condition, specifications and market data. Use a tool to incorporate the info or disregard.
- Email Confirmation: Send a confirmation email with all appointment details.
- SMS Notification: Send SMS notifications for appointment confirmation and reminders.

## Style Guidelines:

- Primary color: Deep carmine (#A92636), reflecting the boldness and importance of the app.
- Background color: Very light desaturated red (#F0E7E8), which provides a contrasting neutral background.
- Accent color: Dark violet (#532563) offers a visually distinct contrast for CTAs and alerts, analogous to the primary color.
- Headline font: 'Playfair', a modern serif with high contrast, gives an elegant feel; body font: 'PT Sans', a modern sans-serif.
- Use clear and modern icons representing different aspects of car information, status and actions.
- Design a clear, step-by-step layout for the voice assistant interface to ensure smooth navigation.
- Use subtle transitions and animations to enhance user experience during interactions.
